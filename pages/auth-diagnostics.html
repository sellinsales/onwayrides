Web Auth Checks

Find where the login is failing.

This page checks the browser environment that Firebase popup and redirect auth depend on: current domain, browser storage, popup behavior, response headers, and backend reachability.

What this can catch

Blocked `sessionStorage`, popup blockers, strict `Cross-Origin-Opener-Policy`, bad API host wiring, and missing public page headers.

What this cannot confirm

Firebase Authorized Domains or WhatsApp Business credentials cannot be read from the browser. Those still have to be checked in the console or Meta dashboard.
